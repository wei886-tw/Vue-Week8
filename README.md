## 使用技術

- 使用 `Vue 3` 和 `create-vue` 構建開發環境
- 選用 `Bootstrap 5` 搭配 `Sass` 進行網頁切版
- 整合 `ESLint Standard` 以保持程式碼風格的一致性
- 運用 `Pinia.js` 作為狀態管理工具，實現跨元件資料的傳遞
